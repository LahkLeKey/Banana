# Steam Storefront Copy Contract

## Purpose

Define the current target copy for the Banana Engineer Steam store page so the
project can align gameplay, disclosure, and launch readiness work against one
canonical storefront surface.

## Contract Version

- Version: 0.1
- Date: 2026-05-22
- Feature: 001-steamworks-release-readiness

## Target Store Identity

- Product name: Banana Engineer
- Genre: Action, Massively Multiplayer, RPG
- Developer: Kyle Halek
- Publisher: Kyle Halek
- Planned release date: Aug 25, 2027

## Current Storefront Claims

- Short description: Banana Engineer is an open source world 2.5D MMOARPG where
  players explore a persistent world, battle in real-time action combat, and
  build their path through progression, crafting, and community-driven systems.
- About copy: Banana Engineer is an open source world 2.5D MMOARPG built around
  persistent progression, real-time action combat, and shared online adventures.
- Player-facing gameplay themes:
  - class-driven builds,
  - gear upgrades,
  - skill combinations,
  - solo play,
  - party-based runs,
  - community-driven expansion.

## Alignment Rules

- Public store copy must remain synchronized with gameplay capabilities, release
  build behavior, and disclosure artifacts.
- Any change to AI service usage, controller support, or system requirements must
  update this contract and the release-readiness artifacts in the same change set.
- This document is the current storefront target, not a promise that all claims
  are already fully implemented.
- Storefront copy should prefer the exact lines above unless a later release
  explicitly changes the published Steam page.
