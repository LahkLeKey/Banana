# Environment Variable Inventory (v1 snapshot)

Sources: `.github/workflows/`, `scripts/`, `docker-compose.yml`.

## Hard runtime contract (must survive into v2 by name)

| Var | Owner | Purpose |
| --- | --- | --- |
| `BANANA_PG_CONNECTION` | Native DAL + ASP.NET | Postgres libpq conn string. Absence triggers `DbNotConfigured` (loud gating). |
| `BANANA_NATIVE_PATH` | ASP.NET interop loader | Directory containing `libbanana_native.so` / `banana_native.dll`. |
| `VITE_BANANA_API_BASE_URL` | React build | Compile-time API base URL injected into Vite bundle. |
| `DATABASE_URL` | Prisma 7 (TypeScript API) | Resolved via `prisma.config.ts` `defineConfig({ datasource: { url: env("DATABASE_URL") } })`. |
| `BANANA_WSL_DISTRO` | Launch scripts | Override Ubuntu distro selection (default order: `BANANA_WSL_DISTRO`, `Ubuntu-24.04`, `Ubuntu`). |
| `BANANA_API_BASE_URL` / `BANANA_API_HOST` / `BANANA_API_PORT` / `BANANA_API_URL` | Compose runtime | Service-to-service resolution inside compose network. |
| `BANANA_LEGACY_API_BASE_URL` / `BANANA_LEGACY_ML_API_BASE_URL` | Migration glue | Points new clients at v1 API during v2 cutover. |
